# Hex-Tic-Tac-Toe
Rules:
* Infinite Grid
* First player plays once. Every other player plays twice
* 6 in a row to win


## Elo Engine Ideas
* 4 in a row with _ on either side
* 4 in a row with _ on both sides
* Trapezium pattern
* 2x 3 in a row with _ on either side
* More Hexes on the outside surrounding other player
